# مستند دانشجویی فرانت‌اند «آزاد ارز»

این مستند برای ارائه به استاد دانشگاه آزاد تهیه شده و هدف آن توضیح معماری، صفحات اصلی و نحوهٔ پیاده‌سازی بخش فرانت‌اند پروژهٔ «آزاد ارز» است. این پروژه با استفاده از **React** و **Vite** توسعه داده شده و از طراحی تیره با کمک **Tailwind CSS** بهره می‌برد. ساختار کلی برنامه به‌صورت تک‌صفحه‌ای پیاده‌سازی شده است.

## فناوری‌ها و ابزارهای مورد استفاده

- **React 18** و **TypeScript** برای ساخت کامپوننت‌های رابط کاربری و افزایش اطمینان کد.
- **Vite** برای اجرای سریع محیط توسعه و ساخت پروژه.
- **Tailwind CSS** برای استایل‌دهی و پیاده‌سازی تم تیره.
- کتابخانه‌های جانبی شامل `react-router-dom` برای مسیریابی، `lucide-react` برای آیکن‌ها و `usehooks-ts` برای استفاده از هوک‌های آماده.
- مدیریت وابستگی‌ها با **pnpm** طبق تنظیمات workspace پروژه.

## ساختار کلی پروژه

- فایل اصلی مسیریابی در `src/App.tsx` قرار دارد و مسیرهای مربوط به صفحهٔ اصلی، ورود، ثبت‌نام، داشبورد، معاملات، کیف پول و پنل مدیریت را مدیریت می‌کند.
- هر صفحه به‌صورت جداگانه در پوشهٔ `src/pages` تعریف شده و کامپوننت‌های قابل استفادهٔ مجدد در `src/components` قرار دارند.
- استایل‌های پایه در فایل `src/index.css` و تنظیمات Tailwind تعریف شده‌اند.

## معرفی صفحات مهم

### صفحهٔ اصلی

- صفحهٔ اصلی از چند بخش مجزا تشکیل شده است: بخش معرفی، نمایش خلاصه‌ای از وضعیت بازار، معرفی امکانات سامانه و بخش دعوت کاربر به ثبت‌نام یا استفاده از خدمات.
- متن‌ها و داده‌های نمونه مانند نام ارزها، قیمت‌ها و درصد تغییرات به زبان فارسی و در فایل‌های ثابت نگهداری می‌شوند.
- دکمه‌های «ثبت‌نام» و «مشاهده بازار» کاربر را به صفحات مربوطه هدایت می‌کنند.

### ناوبری واکنش‌گرا

- منوی بالای سایت با توجه به اندازهٔ صفحه، در حالت دسکتاپ یا موبایل نمایش متفاوتی دارد.
- مسیرهای اصلی مانند خانه، داشبورد، معاملات، کیف پول و پنل مدیریت در منو قابل دسترسی هستند.
- نقش کاربر (برای مثال مدیر سیستم) در نظر گرفته شده تا گزینه‌های مدیریتی فقط برای کاربران مجاز نمایش داده شود.

### صفحهٔ معاملات

- این صفحه از سه بخش اصلی تشکیل شده است: لیست ارزها، نمودار و اطلاعات بازار، و بخش ثبت سفارش.
- اطلاعات نمایشی شامل قیمت، درصد تغییر، حجم معاملات و بیشترین و کمترین قیمت برای ارزهای پرکاربرد به‌صورت نمونه نمایش داده می‌شود.
- کاربر می‌تواند نوع عملیات (خرید یا فروش) و نوع سفارش را انتخاب کرده، مقدار مورد نظر را وارد کند و سفارش را ثبت نماید.
- بخش‌های «دفتر سفارشات» و «معاملات اخیر» داده‌های نمونه را با رنگ‌بندی متفاوت برای خرید و فروش نمایش می‌دهند.

## نحوهٔ اجرا و توسعه پروژه

1. **نصب پیش‌نیازها:** نصب Node.js نسخهٔ ۱۸ یا بالاتر و pnpm نسخهٔ ۹ مطابق تنظیمات پروژه.
2. **نصب وابستگی‌ها:** اجرای دستور زیر در ریشهٔ پروژه:

   ```bash
   pnpm install
   ```

3. **اجرای محیط توسعه:**

   ```bash
   pnpm dev:frontend
   ```

   پس از اجرا، برنامه روی پورتی که Vite اعلام می‌کند (معمولاً 5173) در دسترس خواهد بود.

4. **ساخت نسخهٔ نهایی:**

   ```bash
   pnpm --filter frontend build
   ```

## نکات پایانی برای ارائهٔ دانشگاهی

- داده‌های مالی استفاده‌شده در پروژه صرفاً نمونه هستند و در صورت نیاز می‌توان آن‌ها را واقعی‌تر شبیه‌سازی کرد.
- تمام متن‌ها و رابط کاربری به زبان فارسی طراحی شده‌اند تا تجربهٔ کاربری مناسب‌تری ارائه شود.
- برای نمایش بهتر پروژه، بررسی آن در اندازه‌های مختلف صفحه (موبایل و دسکتاپ) توصیه می‌شود.
